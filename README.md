# nowplaying-sns-cli-mac-fb2k

MacのFoobar2000でもなうぷれをしたい Misskeyへ投稿するやつ

## 動作イメージ

```
$ nowplaying
```

## 依存関係

Homebrewで導入してください。

- jq
- awk (入ってなければ)
- media-control (x64版) ※Rosetta2で動かしてください
